## 📱 Overview

**Jet-Set-App** is a fully designed, user-centric mobile application interface for a premium private jet charter service. This project maps the entire customer journey — from first-time authentication to post-flight feedback — delivering a luxury-aligned, intuitive booking experience.

The design demonstrates a production-ready UI/UX flow ready for front-end development or backend API integration.

## ✨ Key Features

### 🔐 Secure Authentication
- User Login & Sign-up with CNIC and phone number validation
- Smooth onboarding flow

### 🔍 Smart Flight Search
- From / To location pickers
- Date selection, passenger count, and cabin class filters

### 🛩️ Curated Jet Listings
Each jet card displays:
- **Availability status** (Available / Unavailable)
- Seat capacity & range type (Light, Midsize, Heavy)
- Hourly rates & key amenities (Wi-Fi, lavatory, catering)
- Visual indicators for quick comparison

### 📄 Detailed Jet View
- Expanded amenities list
- Prominent "Book Now" CTA
- High-resolution jet imagery

### 💳 Booking & Payment
- Booking summary with breakdown
- Secure payment form (cardholder, card number, expiry, CVV)
- Instant confirmation with unique reference number

### 🎫 Digital Ticket / Boarding Pass
- Passenger information
- Flight reference number
- Jet type, route, date, class, and seat count
- QR-code-ready layout

### 👤 User Profile Management
- Booking history with past and upcoming trips
- Personal details management
- Rating history & post-flight feedback

### ⭐ Post-Journey Rating
- Rate your flight (1-5 stars)
- Submit detailed feedback
